TWICE Pets v1.0.0 Release Notes
这是一个包含 9 只 TWICE 桌宠的小应用。

下载说明（请注意不要下载错了！！）
Windows
推荐普通用户下载：
TWICE-Pets-win11-Setup-1.0.0.exe
安装后会自动创建快捷方式（开始菜单/桌面，取决于系统设置）。

macOS
macOS 分为 Apple Silicon（M 系列）与 Intel 两种架构，请按你的 Mac 选择对应文件：
Apple Silicon（M1/M2/M3/M4）：下载 arm64
Intel Mac：下载 x64
mac 安装时：打开 .dmg，把 TWICE Pets.app 拖到 Applications。
如果提示“无法打开/无法验证开发者”，用右键 TWICE Pets.app -> 打开，或到系统设置里允许。

功能
9 只 TWICE 桌宠可同时显示
支持显示 / 隐藏气泡台词
支持编辑每只宠物的气泡台词
支持自定义每只宠物的名字
支持多档缩放（35% / 50% / 65% / 80% / 100%）
支持鼠标滚轮独立缩放当前宠物
支持陪伴模式
支持托盘 / 菜单栏常驻
支持置顶显示
支持隐藏 Dock / 任务栏图标但保留桌宠

注意
应用可能没有数字签名：
Windows 可能提示“未知发布者”，这是正常现象。
macOS 可能触发 Gatekeeper 提示，需要在系统设置里允许或右键“打开”。
如果升级后图标未刷新，可尝试重装或刷新系统图标缓存。

操作指南（按钮功能）
顶部按钮（宠物控制）
😊 Idle：让桌宠立刻回到待机动作。

💬 Next Bubble：切换到下一句气泡台词。

🐾 Close This Pet：单独关闭当前这只桌宠。

🔍 Scale Pet：按档位循环缩放当前桌宠大小，也支持鼠标滚轮缩放。多个宠物同时显示时，缩放是独立的，光标在哪只宠物窗口上，就缩放哪一只。

👤 Rename Pet：打开名字编辑窗口，可以自定义当前这只宠物的显示名。最多 12 个字，留空并保存可恢复默认名字。

✏️ Edit Bubble Text：打开气泡台词编辑窗口。

🗨️ Toggle Bubbles：显示或隐藏气泡台词。

♡ Companion Mode：开启陪伴模式。开启后鼠标悬停不显示操作面板，减少对桌面操作的干扰。双击桌宠可退出陪伴模式。

📌 Always On Top：让桌宠窗口始终显示在最前。

🙈 Hide All To Tray：隐藏所有桌宠窗口并保留托盘 / 菜单栏运行。

✕ Quit App：退出桌宠程序。

动作按钮（动画）
Right（＞）：播放向右移动动画。

Left（＜）：播放向左移动动画。

Wave（~）：播放挥手动画。

Jump（^）：播放跳跃动画。

Fail（!）：播放失败 / 沮丧动画。

Wait（z）：播放等待动画。

Busy（>>）：播放忙碌动画。

Review（？）：播放查看 / 检查动画。

气泡台词编辑器
New / Edit / Delete：逐条新增 / 修改 / 删除台词。

Save：保存当前桌宠的台词修改。

Restore Default：恢复当前桌宠的默认台词。

Cancel：关闭编辑窗口并取消本次操作。

名字编辑器
点击 👤 Rename Pet 可打开名字编辑器。

Save：保存当前输入的名字。

Default：恢复当前宠物的默认名字。

Cancel：关闭名字编辑器，不保存本次修改。

名字最多 12 个字，避免超出姓名框。

托盘 / 菜单栏功能
多宠物同时显示与隐藏应用图标这两个功能，都是在系统托盘 / 菜单栏图标的右键菜单里开启，不是在主界面按钮里直接开启。

显示所有宠物
右键托盘（win）或左上角菜单栏图标（mac）。

点击 Show All Pets，会显示全部 9 只桌宠。

选择显示哪些宠物
右键托盘（win）或左上角菜单栏图标（mac）。

打开 Pets On Desktop。

勾选你想同时显示的宠物，勾选几只就会同时出现几只。

取消某一只的勾选，就只会关闭那一只。

隐藏应用图标但保留桌宠
macOS：右键菜单里勾选 Hide Dock Icon，Dock 图标会隐藏，但桌面宠物会继续保留。

Windows：右键菜单里勾选 Hide Taskbar Icon，任务栏图标会隐藏，但桌面宠物会继续保留。

需要恢复图标时，再次从托盘 / 菜单栏图标菜单取消勾选即可。

补充说明
鼠标移动到桌宠上方会显示名字与操作面板。

按住桌宠本体可拖动位置。

单击桌宠会切换到下一句气泡台词。

如果不想开机自启，在隐藏栏里面关掉 Open at Login 选项即可。
