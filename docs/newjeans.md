# NewJeans Pets v1.0.1 Release Notes

## 下载说明

- Windows：下载 `NewJeans.Pets.Setup.1.0.1.exe`
- macOS Apple Silicon / M 系列：下载 `NewJeans.Pets-1.0.1-arm64.dmg`
- macOS Intel：下载 `NewJeans.Pets-1.0.1-x64.dmg`

## 使用说明

- 启动后桌宠会出现在桌面，可以拖动到喜欢的位置。
- 托盘菜单可以管理显示、隐藏和退出。
- 如果 Windows 安全提示拦截，请选择保留，并确认这是你自己发布的安装包。
