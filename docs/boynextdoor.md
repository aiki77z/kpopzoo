# BOYNEXTDOOR Pets v1.0.3

这是一个包含 6 只 BOYNEXTDOOR 主题桌宠的小应用。

## 下载说明

- **BOYNEXTDOOR-Pets-win11-Setup-1.0.3.exe**  
  推荐普通用户下载。安装后会自动创建快捷方式。

- **BOYNEXTDOOR.Pets-1.0.3-arm64.dmg**  
  适用于 Apple Silicon（M 系列）Mac，下载后打开 `.dmg` 并拖入 `Applications`。

- **BOYNEXTDOOR.Pets-1.0.3-x64.dmg**  
  适用于 Intel Mac，下载后打开 `.dmg` 并拖入 `Applications`。

## 本次更新

- 新增更小的桌宠缩放档位，缩放范围调整为 35% / 50% / 65% / 80% / 100%。缩放可通过 🔍 按钮操作，也可通过滚轮操作。
- 拖拽逻辑优化：鼠标按住桌宠本体即可直接拖动。
- 操作面板改为显示在桌宠下方，不再影响桌宠本体位置。
- 移除详情面板，界面结构更简洁。
- 宠物名称只在鼠标悬停桌宠时显示。
- 气泡框样式重新调整：固定在头顶上方，宽度会根据字数变化，最多显示两行。
- 气泡文字字号会跟随桌宠缩放同步变化。
- 气泡文字编辑窗口改为列表式管理，支持逐条新增、编辑、删除和恢复默认。
- 动画切换逻辑优化：点击动作后会持续播放，直到切换到下一个动作。
- 新增陪伴模式：开启后不会显示操作面板，减少对桌面操作的干扰；双击桌宠可退出陪伴模式。
- 陪伴模式下窗口交互范围进一步缩小，尽量减少遮挡桌面点击。

## 功能

- 6 只桌宠可切换
- 可编辑气泡文字
- 可切换动作
- 支持多档缩放
- 支持陪伴模式
- 支持托盘
- 支持置顶显示

## 注意

因为应用暂时没有数字签名，Windows 可能会提示“未知发布者”，这是正常现象。  
如果升级后图标显示没刷新，可以尝试重新安装或刷新系统图标缓存。

## 操作指南

- **😊 Idle**：让宠物立刻回到待机状态。
- **💬 Next Bubble**：切换到下一句气泡文字。
- **🐾 Next Pet**：切换到下一只桌宠。
- **🔍 Scale Pet**：按档位循环缩放宠物大小。
- **✏️ Edit Bubble Text**：打开气泡文字编辑窗口。
- **🗨️ Toggle Bubbles**：显示或隐藏气泡文字。
- **♡ Companion Mode**：开启陪伴模式。开启后鼠标悬停桌宠不会显示操作面板，双击桌宠可退出。
- **📌 Always On Top**：让桌宠窗口始终显示在最前面。
- **🙈 Hide To Tray**：隐藏主窗口并保留托盘运行。
- **✕ Quit**：退出桌宠程序。
- **> Right**：播放向右移动动画。
- **< Left**：播放向左移动动画。
- **~ Wave**：播放挥手动画。
- **^ Jump**：播放跳跃动画。
- **! Fail**：播放失败/沮丧动画。
- **z Wait**：播放等待动画。
- **>> Busy**：播放忙碌动画。
- **? Review**：播放查看/检查动画。
- **New / Edit / Delete**：在气泡文字编辑窗口中逐条新增、修改、删除台词。
- **Save**：保存当前桌宠的气泡文字修改。
- **Restore Default**：恢复当前桌宠的默认气泡文字。
- **Cancel**：关闭当前编辑窗口而不继续操作。

## 补充说明

- 将鼠标移动到桌宠本体上方即可显示名字和操作面板。
- 按住桌宠本体即可拖动位置。
- 点击桌宠会切换下一句气泡文字。
- 陪伴模式下不会展示操作面板，适合长时间放在桌面陪伴使用。
